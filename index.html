使用方法

简单几步即可开始使用我们的镜像服务

1

使用GitHub资源

将GitHub资源链接转换为镜像链接：

// 原始链接
https://github.com/user/repo/blob/main/file.js

// 镜像链接
https://jsdelivr.nakeu.cn/gh/user/repo@main/file.js
2

在HTML中使用

在HTML文件中直接引用资源：

<!-- 引入jQuery -->
<script src="https://jsdelivr.nakeu.cn/gh/jquery/jquery@3.6.0/dist/jquery.min.js"></script>

<!-- 引入Bootstrap CSS -->
<link rel="stylesheet" href="https://jsdelivr.nakeu.cn/gh/twbs/bootstrap@5.1.3/dist/css/bootstrap.min.css">
3

使用转换工具

使用我们的GitHub链接转换工具快速生成镜像链接：

// 输入GitHub资源URL
https://github.com/user/repo/blob/main/file.js

// 工具自动生成镜像链接
https://jsdelivr.nakeu.cn/gh/user/repo@main/file.js

SRI哈希生成工具

生成子资源完整性(SRI)哈希值，增强资源安全性

什么是SRI？

子资源完整性(SRI)是一种安全功能，允许浏览器验证获取的资源（例如从CDN获取的）是否被篡改。

在HTML中使用：

<script src="https://example.com/example.js" integrity="sha384-oqVuAfXRKap7fdgcCY5uykM6+R9GqQ8K/uxy9rx7HNQlGYl1kPzQho1wx4JwY8wC" crossorigin="anonymous"></script>

使用SRI可以防止恶意第三方篡改资源内容，提高网站安全性。
